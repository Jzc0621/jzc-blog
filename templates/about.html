项目

基于 RA6M5 与 TinyML 的智能脉搏监测系统

以瑞萨 RA6M5 (Cortex-M33 @200MHz) 为核心，PPG 传感器采集脉搏波， 在 MCU 端运行自研 1D-CNN 进行心律实时分类。 TFT 本地显示 + PC 端 Python 看板双屏同步可视化。

技术栈：C · FSP 6.4.0 · 纯 C 神经网络推理 · Python · e2 studio
