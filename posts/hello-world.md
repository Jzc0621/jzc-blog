## 为什么建这个博客

一直想找一个安静的地方写东西。不需要评论数、点赞、阅读量，只是想把自己学到的、想到的、遇到的事情记下来。

这个博客的视觉风格参考了《动物森友会》游戏界面——温暖的色调、圆润的按钮、不紧不慢的氛围。希望在这写东西的时候也能保持这种心情。

## 会写什么

- **技术笔记**：Python、Flask、以及平时踩的坑
- **生活记录**：徒步、咖啡、遇见的有趣事物
- **随手便签**：不成篇的碎片想法，放在便签墙那边

## 技术栈

这个博客本身也很简单：

```python
# 核心依赖
import flask
import markdown
import frontmatter

# 没有数据库，文件系统就是存储
# 没有登录，写就完了
```

用 Markdown 文件存储文章，Flask 渲染页面，纯 CSS 模拟动森风格。

## 慢慢来

不追求更新频率，想写了就写，不想写就放着。

就像岛上钓鱼一样——有没有鱼上钩不重要，重要的是坐在码头上的那段时间。
